# 로또 프로그램
**입력:** 
1. 로또를 구매할 금액
2. 로또 당첨 번호
3. 보너스 번호  
-----------------
**출력:** 
1. 구매한 로또의 수량과 번호
2. 당첨된 개수
3. 수익률  
-----------------

### 기능 구현 필요 목록:
- [X] 로또 클래스 구현
    #### 로또 클래스로 생성된 객체에는 복권 1장의 번호가 담겨있음.
-----------------
    예) Lotto ticket = new Lotto(랜덤 번호);
    &rightarrow;ticket.numbers 는 List<Integer> 로 이루어진 랜덤 번호를 저장하게 됨.
-----------------
    메소드 목록
    1. validateSize()
        숫자의 개수가 6을 넘어서는지 확인
    2. validateOverlap()
        숫자가 중복되지않는지 확인
    3. validateOutOfRange()
        숫자가 1~45 범위 안에 있는지 확인
    4. getLottoNumbers()
        나중에 Lotto의 번호를 받아올 필요가 있음.
